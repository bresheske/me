angular route animations made easy

August 16th, 2017


So I found a super-simple method of getting animations working in angular. Fair warning: this is not a very elegant solution, but I don't feel like it's breaking any rules or anything.

1. Check out routing animations, the right way

Google it. After following some tutorials, I personally found the solutions pretty complicated and full of non-standard code.

2. Add ScrollRevealJS

ScrollRevealJS link.

Type this code into your command line environment:

npm install --save scrollreveal

Now you need to add it to your .angular-cli.json file. It should look something like this:

"scripts": [
        "../node_modules/scrollreveal/dist/scrollreveal.js"
      ],
